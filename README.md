# 📱 Responsive NavBar

This project showcases a responsive navigation bar that adapts to different screen sizes, featuring a hamburger menu for mobile devices.

## 🌐 Live Demo

Check out the live demo of the responsive navbar [Here](https://66c163814e54890b8b8ced44--dinesh-responsive-navbar.netlify.app/).

## ✨ Features

- 📱 **Responsive Design**: The navigation bar adjusts for both desktop and mobile views.
- 🍔 **Hamburger Menu**: Displays a collapsible menu on smaller screens.
- 🎨 **Customizable Styling**: Easy to modify the appearance to fit your design needs.

## 🛠️ Technologies Used

- **HTML5**: Provides the structure of the webpage.
- **CSS3**: Used for styling and creating a responsive layout.
- **JavaScript**: Handles the toggling of the navigation menu.

## 🚀 How to Use

1. 📥 **Clone the repository**:
   ```bash
   git clone https://github.com/dinesh-fullstackwebdeveloper/responsive-navbar.git
   ```
2. 🌐 **Open `index.html`** in your web browser to view the navigation bar.
3. 🎨 **Customize** the styles and functionality by editing `style.css` and `script.js` as needed.

## 📁 Project Structure

```
.
├── index.html
├── style.css
└── script.js
```

- **index.html**: Contains the HTML structure for the navbar and menu.
- **style.css**: Defines the styles for the header, navbar, and responsive behavior.
- **script.js**: Manages the toggle functionality for the hamburger menu.

## 🎨 Customization

- ✨ **Text Content**: Modify the text inside the `<div class="logo">` and navigation items in `index.html`.
- 🎨 **Colors and Layout**: Adjust the colors, padding, and layout in `style.css`.
- ⏱️ **Button and Menu Behavior**: Modify the JavaScript function in `script.js` to change how the menu toggles.
